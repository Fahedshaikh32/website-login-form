# 🌐 Website Login Form

A simple and clean login page developed using HTML and CSS. This project focuses on creating a modern user interface with proper layout, styling, and responsiveness.

---

## 📌 Features
- User-friendly login interface  
- Clean and modern UI design  
- Responsive layout  
- Styled form elements with CSS  
- Background effects and smooth design  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  

---

## 📖 Project Description
This project was developed to understand the basics of front-end development. It includes designing a login form with proper structure, styling, and alignment. The focus was on improving UI design skills and creating a visually appealing webpage.

---

## 🚀 How to Run
1. Download or clone the repository  
2. Open the project folder  
3. Run the `index.html` file in any web browser
